# ta501
Input Selector Board

## Purpose
Basic input board for **TuchAudio 303 integrated amplifier**, a.k.a. _The Bedroom Amp_, together with:

- **TuchAudio 103:** Phono pre-amp.
- **TuchAduio 202:** compact power amp.
- A generic **12V/2A power supply**

## Inputs

- **Phono:** 3.6 mV / 110 kohm. (via `103 pre-amp`)
- **Line:** 180 mV / 900 kohm.
- **Tape Loop:** with RCA (180 mV/47 k) and DIN (0.48 V/16.5 k) connectors.

## Outputs

- Speaker: **~1.5 W RMS** over 8 ohm.
- Tape Rec (RCA): 180 mV
- Tape Rec (DIN): 5.3 mV over 10 kohm
